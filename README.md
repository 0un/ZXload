# ZXload 📼

**ZXload** — это легковесный Python-скрипт для Android (в среде [Termux](https://termux.dev)), предназначенный для загрузки игр и программ в **ZX Spectrum** напрямую со смартфона через аудиокабель.

Скрипт на лету генерирует математически идеальный аудиосигнал с правильными таймингами, инвертирует фазу для стабильного распознавания Спектрумом и выводит прогресс-бар в консоль.

---

## Основные возможности 🚀

* 📦 **Три формата в одном:** Воспроизведение файлов `.tap`, `.tzx` и `.wav` одной командой.
* 🛠️ **Абсолютная автономность:** Не требует тяжелых Python-библиотек, которые часто не компилируются на мобильных ARM-процессорах.
* 📊 **Интерактивный прогресс:** Выводит точный процент загрузки, прошедшее и **оставшееся время** до конца передачи.
* 🛡️ **Защита от сбоев (`R Tape loading error`):** 
  * Встроена автоматическая инверсия фазы сигнала для уверенного старта.
  * Добавлена секундная пауза в начале для стабилизации входных конденсаторов ZX Spectrum.
* ⚡ **Низкое потребление памяти:** Файлы генерируются кусками и не перегружают ОЗУ смартфона.

## Инструкция по установке в Termux (Android) 📲

Откройте Termux на вашем Android-устройстве и выполните следующие команды по очереди:

### 1. Обновление системы и установка окружения
```bash
pkg update && pkg upgrade -y
pkg install python sox nano -y
```

### 2. Предоставление доступа к памяти телефона
Чтобы скрипт мог читать файлы из вашей папки «Загрузки», дайте Termux права на чтение памяти:
```bash
termux-setup-storage
```
*В появившемся системном окне Android нажмите **«Разрешить»**.* Теперь ваши загрузки доступны по пути `~/storage/downloads/`.

### 3. Установка скрипта
Создайте файл скрипта:
```bash
nano load.py
```
Скопируйте код скрипта `load.py`, зажмите экран в Termux, выберите **Paste** и сохраните файл (`Ctrl + O`, затем `Enter`, для выхода `Ctrl + X`).

Сделайте файл исполняемым:
```bash
chmod +x load.py
```

## Как пользоваться 🎮

1. Соедините аудиокабелем (3.5 мм Jack) разъем наушников смартфона и разъем **EAR** вашего ZX Spectrum.
2. На Спектруме введите команду `LOAD ""` и нажмите `ENTER`.
3. В Termux запустите проигрывание нужного файла:

```bash
# Воспроизведение TAP файла (из папки со скриптом)
./load.py game.tap

# Воспроизведение TZX файла (из папки Загрузки телефона)
./load.py ~/storage/downloads/game.tzx

# Воспроизведение готового аудиофайла WAV
./load.py game.wav
```

Во время воспроизведения утилита `sox` автоматически покажет статус:
```text
In:45.2% 00:02:15.00 [00:01:45.00] Out:967k  [=======>            ]
```
Где в квадратных скобках `[00:01:45.00]` отображается **время, оставшееся до конца загрузки**. Для отмены воспроизведения нажмите `Ctrl + C`.

Если нужно поставить на паузу нажимаем **Ctrl+Z**, а для возобновления вводим команду `fg`.

## Советы по стабильной загрузке 💡

Если вы получаете ошибку `R Tape loading error` в процессе загрузки:
1. **Громкость:** Установите громкость на смартфоне на уровень **80–90%**. Не ставьте 100%, так как усилитель телефона начнет хрипеть и искажать форму импульсов.
2. **Улучшайзеры звука:** Полностью отключите в настройках Android любые эквалайзеры, Dolby Atmos, Mi Sound или аудио-эффекты. Они сглаживают прямоугольную волну, превращая её в кашу для Спектрума.
3. **Питание:** Обязательно **отключите зарядное устройство** от смартфона на время загрузки. Импульсные зарядки создают наводки по общему проводу (земле).

## English Short Description 🇬🇧

**ZXload** is a lightweight Python script designed for Android (Termux environment) to load games into a real **ZX Spectrum** computer via an audio cable. 

### Features:
* Supports `.tap`, `.tzx`, and `.wav` formats natively.
* Zero external Python dependencies (no `numpy` or `sounddevice` needed).
* Displays live progress bar, percentage, elapsed, and **remaining time** using `sox`.
* Auto phase inversion and a 1-second initial delay for reliable loading without `R Tape loading error`.

### Quick Start:
1. Install dependencies in Termux: `pkg install python sox -y`
2. Grant storage access: `termux-setup-storage`
3. Make executable: `chmod +x load.py`
4. Run: `./load.py game.tap` (or `.tzx` / `.wav`)
